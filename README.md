## Web

### Environment Variable
Rename the `.env.example` file to `.env` and update the `FILE_Path` and `AUTH_SECRET` variables .

### Install Packages
```angular2html
bun install
```
### Create Database
```angular2html
bun run db:push
```
### Start Application
```angular2html
bun run dev
```
